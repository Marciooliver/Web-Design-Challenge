Summary: Latitude vs. X

In this project, we are aiming to check the changes in the weather behavior as you get closer to the equator. In order to formulate the analysis, we chose four distinct variables used to describe the atmospheric conditions - cloudiness, humidity, temperature, and wind speed.

We pulled data from the OpenWeatherMap API to assemble a dataset with over 500 points randomly distributed around the globe. Here you find the visual representation of each data set using Matplotlib, along with a description of trends and correlations observed.
